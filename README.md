# lmaobox
A decompiled free version of LMAOBox, reconstructed from compiled binaries. This repository provides readable source code for analysis, debugging, and educational purposes. Note: This decompiled code may differ from the original source and is provided as-is.

Tools used:

dnSpy (For the loader, and the encoded base64 string.)
UPX (For unpacking the packed string.)
IDA Free (For ASM.)
ASM to C++ (Custom tool, to turn it in to a pseudo C++ program.)
